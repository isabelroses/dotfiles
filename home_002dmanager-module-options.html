3.4 programs.gtklock.config.modules

A list of gtklock modulesto use. Can either be packages, absolute paths, or strings.

Type: list of (package or string)

Default: [ ]

Example: 

[
  "/nix/store/84qwdf0khj9hyy78w02bn0j65kn669zz-gtklock-powerbar-module-2.0.1/lib/gtklock/powerbar-module.so"
  "/nix/store/dzvl08gbg5lakh476lsigzl907n4yslp-gtklock-playerctl-module-2.0.1/lib/gtklock/playerctl-module.so"

3.5 programs.gtklock.config.style

The css file to be used for gtklock.

Type: string or path

Default: ""

Example: 

pkgs.writeText "gtklock-style.css" ''
  window {
    background-size: cover;
    background-repeat: no-repeat;
    background-position: center;
  }
''

Declared by: - modules/extra/home-manager/gtklock

3.6 programs.gtklock.extraConfig

Extra configuration to append to gtklock configuration file. Mostly used for appending module configurations.

Type: null or (attribute set)

Default: 

{
  countdown = {
    countdown = 20;
    countdown-position = "top-right";
    justify = "right";
  };
}

Example: 

countdown = {
  countdown-position = "top-right";
  justify = "right";
  countdown = 20;
}
